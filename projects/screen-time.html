Screen Time Visualizer

Analyzes TV series footage to detect character presence, then turns it into charts and animated timelines.

GitHub Back to list
Python Data viz Automation
Snapshot
Pulls video frames, tags character appearances, and aggregates per-episode stats. Outputs bar charts and animated visuals showing who appears when. Works best with playlists/season links; handles scraping and processing.
Overview

What it does

Flow

Ingest YouTube playlist or local episodes. Detect faces per frame, map to characters, aggregate screen time per episode. Generate plots (static + animated) for quick sharing.

Notes

Built in Python; uses standard plotting stack. Easy to rerun on new shows—swap playlist and labels. Add screenshots/GIFs below for quick demos when you have them handy.
Highlights

Features

Data collection

Automatically fetches episodes via playlist link; ready to swap sources.

Processing

Calculates percentage presence per episode and per season.

Visualization

Builds static charts and animations to show character timelines.
