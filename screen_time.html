About the Project

The "Screen Time of Series" project is a data analysis and visualization project that focuses on analyzing the screen time of different characters in a TV series. The project involves processing facial recognition data to determine the presence of characters in each episode and then visualizing their appearance percentages.

Key Features

  • Data Collection

    Collects data on character appearances in each episode of a TV series. Right now we can give YouTube playlist link, and we can get all necessary data from that easily.

  • Data Processing

    Processes the collected data to calculate the percentage of appearance for each character in each episode.

  • Data Visualization

    Creates dynamic bar plots that showcase the character appearances throughout the series.

  • Animation

    Utilizes Matplotlib's animation features to display character appearances in multiple episodes.

  • Customization

    Users can select specific characters and episodes to visualize, providing an interactive experience.
